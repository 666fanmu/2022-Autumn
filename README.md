# 2022-Autumn
* Gamecore 2022年秋季考核  
## 前言
* 游戏制作不能只看到快乐的一面“__高薪，创意，设计，灵感__”    
* 也要看到海量的“__练习 ，思考，困惑，探索__”   
* 只有当翻越之后你才能看到其中快乐的一面。   
* 而你能否成功翻越，最大的因素是：“__能否从你的工作内容本身感受快乐与成就！__”
